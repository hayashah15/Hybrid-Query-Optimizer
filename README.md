# Hybrid-Query-Optimizer
Adaptive Execution Planning for Hybrid Relational-Vector Workloads using PostgreSQL and pgvector
## Team Members

- Haya Shah (Project Lead)
- Dipesh Adhikari
- Uday Kumar Reddy

## Project Description

This project implements an adaptive execution planning system for hybrid relational-vector workloads using PostgreSQL and pgvector.

The system combines traditional SQL queries with semantic vector search and automatically selects an efficient execution strategy.
